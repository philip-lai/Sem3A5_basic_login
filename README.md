# 安裝說明
#### 1. 開啟終端機，將專案下載至本機端

      git clone https://github.com/philip-lai/Sem3A5_basic_login.git
#### 2. 切換檔案路徑至專案資料夾，並安裝套件

      cd Sem3A5_basic_login
      npm install
#### 3. 執行專案
      npm run dev
      

# 使用工具
* visual studio
* Express
* Express-handlebars
